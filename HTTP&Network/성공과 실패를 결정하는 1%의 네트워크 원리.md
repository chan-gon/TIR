# 성공과 실패를 결정하는 1%의 네트워크 원리

## 구입 동기

웹 프로그래밍 기초 지식인 네트워크의 구조와 동작 원리를 파악하기 위해 구입했다. 기초를 탄탄하게 다지고 싶었다.

## 소감

클라이언트에서 리퀘스트 메시지를 보내고 서버에서 리스폰스 메시지를 보내는 전 과정을 6단계로 분해하여 분석하고 있다.  
1장 클라이언트에서 메시지를 보내고, 6장 웹 서버에서 클라이언트의 요청을 받아 리스폰스를 전달하는 두 파트만 이해가 쉬웠고 나머지 2장 ~ 5장은 50%도 이해하지 못했다. 본인이 네트워크 개념 및 관련 용어 정리가 명확하지 않은 점, 실습 및 실무 경험이 전무한 점 때문에 책에서 설명하는 개념의 흐름을 그려내기 어려웠다고 생각한다. 곁에 두고 틈틈이 들여다봐야겠다.

## 특히 좋았던 부분

클라이언트 요청에서 응답을 전달받는 수 초간의 작업 내용을 꼼꼼히 들여다볼 수 있어서 좋았다.
